# Turon Zakas Ultimate - Mukammal Boshqaruv Tizimi

Ushbu loyiha taksi va yetkazib berish xizmatlarini to'liq avtomatlashtirish, hodimlar ishini tahlil qilish va baholash tizimi bilan integratsiya qilingan yagona platformadir.

## 🚀 Texnologiyalar
- **Backend**: Node.js, Express, Socket.io
- **Database**: MongoDB
- **Bot**: Telegram Bot API (Mijoz va Hodimlar uchun)
- **Frontend**: HTML5, Vanilla CSS, Bootstrap 5

## 🛠 O'rnatish

1.  **Clone qiling**:
    ```bash
    git clone [repo-url]
    cd turon-zakas-ultimate
    ```
2.  **Kutubxonalarni o'rnating**:
    ```bash
    npm install
    ```
3.  **Environment Variables (`.env`)**:
    `.env` faylini yarating va quyidagilarni kiriting:
    ```env
    MONGODB_URI=sizning-mongodb-manzilingiz
    TELEGRAM_BOT_TOKEN=sizning-bot-tokeningiz
    ADMIN_PASSWORD=admin-panel-paroli
    PORT=5000
    ```
4.  **Ishga tushirish**:
    ```bash
    npm start
    ```

## 📈 Loyiha Imkoniyatlari
- **Admin Panel**: Buyurtmalarni biriktirish, xodimlarni reyting bo'yicha saralash va tahlil qilish.
- **Baholash Tizimi**: Mijozlar xizmatni 1-5 ballgacha baholashlari.
- **Real-vaqt Xabarnomalari**: Buyurtma holati o'zgarganda mijoz va xodimga xabar borishi.
